Course Description

  1. Java - Java is a high-level, object-oriented programming language and computing platform originally developed by Sun Microsystems (now owned by Oracle) and released in 1995. It is defined by the "Write Once, Run Anywhere" (WORA) principle, allowing compiled code to run on any device with a Java Virtual Machine (JVM) without recompilation.

  2. Python - Python is a high-level, general-purpose programming language known for its readability, simplicity, and versatility. Designed by Guido van Rossum and first released in 1991, it emphasizes code clarity through significant indentation and supports multiple paradigms, including object-oriented, procedural, and functional programming.

Learning Outcomes

  • Understand programming fundamentals.
  • Develop problem-solving skills.
  • Create applications using Java and Python.
  • Gain hands-on coding experience.

Concepts Covered

  1. Variables and Data Types
  2. Conditional Statements
  3. Loops
  4. Functions and Methods
  5. Object-Oriented Programming
  6. File Handling
